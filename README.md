# Mc-2.0-Legacy
Mc 2.0 Legacy — Мини-Лаунчер для Майнкрафта 2.0 (April Fools)

Это новый Мини-Лаунчер, который даст вам поиграть в Minecraft 2.0 (Первоапрельское обновление) Без всяких мучений с остальным, Мини-Лаунчер просто нужно скачать, и играть!

Скачать Лаунчер очень просто! Распакуйте его в какую либо папку, и нажимайте "Mc 2.0 Legacy.bat"

## 💻 Системные Требования для Игры и Мини-Лаунчера
Данный проект потребляет очень мало ресурсов, так как версия игры довольно старая (1 апреля 2013 года)

Минимальные:
- Windows 7, 8, 8.1
- 256 МБ ОЗУ заданный в Мини-лаунчере для игры при минимальных настройках графики. 1.5 ГБ ОЗУ в Устройстве.
- Место на диске: 300 МБ.
- Процессор: Intel Pentium D, AMD Athlon 64.
- Видеокарта: Nvidia GeForce 9600 GT, AMD Radeon HD 2400

Рекомендуемые:
- Windows 7 или выше
- 600 МБ ОЗУ заданный в Мини-лаунчере для игры при средних настройках графики. 3 ГБ ОЗУ в Устройстве
- Место на диске: 350 МБ
- Процессор: Intel Core 2 Duo E8400, AMD Athlon II X2 250
- Видеокарта: Nvidia GeForce GT 240, AMD Radeon HD 5570
