# Mc-2.0-Legacy
Mc 2.0 Legacy — Мини-Лаунчер для Майнкрафта 2.0 (April Fools)

Это новый Мини-Лаунчер, который даст вам поиграть в Minecraft 2.0 (Первоапрельское обновление) Без всяких мучений с остальным, Мини-Лаунчер просто нужно скачать, и играть!

<img width="793" height="103" alt="mc20legacy" src="https://github.com/user-attachments/assets/15118a37-316a-4a2e-83bc-3e33aff7f272" />

Java не требуется для запуска игры, так как она уже предустановлена для удобства.
Скачать Лаунчер очень просто! Распакуйте его в какую либо папку, и нажимайте "Mc 2.0 Legacy.bat"

Скачать можно тут https://github.com/Gerser115/Mc-2.0-Legacy/releases/tag/Version
